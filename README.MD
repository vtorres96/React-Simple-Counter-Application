 # Simple application developed using:
 ```sh  
 $   - functional component 
 $   - arrow functions

 ```

## How to install packages (backend)

### after cloning the repository enter the backend folder

```sh
$ cd backend
```

### yarn

```sh
$ yarn
$ yarn dev
```

### npm

```sh
$ npm install
$ npm start
```

## How to run application

### yarn

```sh
$ yarn dev
```

### npm

```sh
$ npm start
```

### Listening port on 3000

```sh
$ http://localhost:3000/
```
